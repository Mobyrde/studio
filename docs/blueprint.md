# **App Name**: Neon Slither

## Core Features:

- Mouse-Controlled Movement: Control the snake's direction by moving the mouse cursor around the canvas.
- Food Consumption and Growth: The snake grows longer as it consumes food scattered throughout the game world.
- Dynamic Bot Opponents: Multiple AI-controlled bots that move around the game world, providing a dynamic challenge for the player. These bots avoid walls and change directions randomly to add variety to the gameplay.
- Score and High Score Tracking: Keep track of the player's score and high score during gameplay.
- Game Over Detection: Detect when the snake collides with the walls of the game world.
- Food Color Palette Generation: Employ a tool that uses generative AI to decide on the colors used in the display of the food

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) for the snake, evoking a sense of futuristic energy.
- Background color: Dark Navy (#1A1A2E) to create a high-contrast backdrop, enhancing the visibility of the neon elements. (HSL 240, 48%, 14%)
- Accent color: Cyan (#00FFFF) to highlight the world border
- Body and headline font: 'Space Grotesk' sans-serif for menus and in-game text.
- Minimalist, geometric icons for UI elements.
- Smooth transitions and subtle animations to enhance user experience, like a neon pulsing effect on the snake.
